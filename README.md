# 名義SPOT進捗チェッカー

Excel ファイルをドラッグ＆ドロップするだけで、「未制作」「制作中」「確認中」の行を抽出し、名義別の進捗を確認できるアプリです。

## できること

- Excel（.xlsx / .xls）をドラッグ＆ドロップで読み込み
- 「製作状況」シート（なければ先頭シート）を使用
- 制作状況が「未制作」「制作中」「確認中」の行のみ抽出
- 申込社名の「株式会社」を除去して整形
- 結果をコピー可能なテキストとして出力

## 仕様（出力フォーマット）

```
＜未制作＞
6/6,7：SAKAE SP-RING 2026（ZIP-FM）
…
＜確認中＞
…
```

## 必要な列

- 制作状況
- アーティスト名
- 申込社
- 公演日
